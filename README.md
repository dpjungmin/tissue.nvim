# tissue.nvim

A dark theme for neovim.


## Installation

[packer]

```lua
use 'dpjungmin/tissue.nvim'
```

## Usage

```lua
-- Lua
vim.cmd 'colorscheme tissue'
```

```vim
" Vim Script
colorscheme tissue
```
